# Quarashi Chat

Use the webRTC and svelte.

Signaling Sever : Socket.io

WebRTC connection : Coturn sever(turn and stun server)

## Installation
1. Backend
Open the terminal
```
npm install
```
or 
```
yarn install
```

2. Fronted
Open the terminal
```
cd client/
npm install or yarn install
```

## Running
1. Backend
```
npm run dev or yarn dev
```

2. Frontend
```
cd client
npm run dev or yarn dev
```
